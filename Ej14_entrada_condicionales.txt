# Entrada de datos, evaluación de divisibilidad y paridad.

EJERCICIO 14.    
    Takes two integers as input from the user
    Calculates and prints:
        The remainder when the first number is divided by the second
        Whether the sum of the two numbers is even or odd
        Whether the first number is divisible by the second (no remainder)

Example Input:
17
5

Expected Output:
Remainder: 2
Sum is odd: 1
First is divisible by second: 0

## Lo aprendido:
- Cómo usar scanf() para leer datos del usuario.
- Cómo detectar si un número es par o impar con % 2.
- Cómo comprobar si un número es divisible por otro.
- Por qué una expresión como x % 2 == 0 || x % 2 != 0 siempre es cierta, y cómo eso puede llevarte a un error lógico aunque el programa compile.
